IF LEADER WINS

LEADER

  • receives 1000 tokens
  • proceeds to the Leaders' Contest (if there is one).

EACH FOLLOWER

  • receives 0 token
  • proceeds to the Help/Sabotage Decision stage (if there is one).

IF LEADER LOSES

LEADER

  • receives 0 token
  • proceeds to the Help/Sabotage Decision stage (if there is one)

EACH FOLLOWER

  • receives 0 token
  • proceeds to the Followers' Contest (if there is one)

Neither follower receives any tokens but pays his investment cost.

WINNER

  • becomes the new leader
  • proceeds to the Leaders' Contest

LOSER

  • continues as a follower
  • proceeds to the Help/Sabotage Decision stage
